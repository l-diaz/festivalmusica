# Festival Música

Sitio web de ejemplo para un festival de música desarrollado como proyecto de aprendizaje de HTML5, CSS3, JavaScript, SASS y Gulp.

## Estructura del proyecto

- `index.html` - página principal del sitio.
- `src/` - archivos fuente.
  - `src/scss/` - estilos escritos en SASS.
  - `src/js/` - código JavaScript fuente.
  - `src/img/` - imágenes del proyecto.
- `build/` - archivos generados y listos para producción.
  - `build/css/app.css` - CSS compilado desde SASS.
  - `build/js/script.js` - JavaScript procesado.
- `video/` - videos usados en el diseño.
- `gulpfile.js` - tareas de Gulp para automatizar el flujo de trabajo.
- `package.json` - dependencias y scripts de NPM.

## Tecnologías

- HTML5
- CSS3
- JavaScript
- SASS
- Gulp
- NPM

## Comandos

- `npm install` - instala dependencias.
- `npm run sass` - observa el directorio `src/scss` y compila los cambios en `build/css`.
- `npm run dev` - ejecuta Gulp.

## Uso

1. Clona o descarga el proyecto.
2. Ejecuta `npm install`.
3. Inicia el proceso de desarrollo con `npm run dev` o `npm run sass`.
4. Abre `index.html` en el navegador para ver el sitio.
